# M4 — Dagger Build Pipeline

## Goal

Build OCI images from the example workshop using Dagger. Each step image contains all metadata baked into `/workshop/` and the backend binary. This is the first milestone where you can actually run the full stack in Docker.

## Prerequisites

- M1 complete (`examples/hello-linux/` exists)
- M2 complete (`pkg/workshop/` compiles, shared library works)
- M3 complete (`backend/` compiles)

## Dagger Module Setup

The Dagger module has its **own** `go.mod` at `dagger/go.mod`, separate from the repo root `go.mod`. The Dagger SDK is NOT in the root `go.mod`.

### `dagger/dagger.json`

Initialize with:
```bash
cd dagger && dagger init --name workshop-builder --sdk go
```

This creates `dagger.json`. Contents look like:
```json
{
  "name": "workshop-builder",
  "sdk": "go",
  "engineVersion": "v0.18.x"
}
```

Use whatever version `dagger init` produces.

### `dagger/go.mod`

Generated by `dagger init`. Will contain the Dagger SDK dependency. Do not hand-write.

## `dagger/main.go`

This is the main Dagger module. All build functions live here.

```go
// Package main is the Dagger module for building workshop-builder components.
package main

import (
    "context"
    "fmt"
    "path/filepath"

    "dagger.io/dagger"
    "github.com/asocpro/workshop-builder/pkg/workshop" // imported from host repo
)

// WorkshopBuilder is the Dagger module.
type WorkshopBuilder struct{}

// Test runs all Go tests in the repository.
func (m *WorkshopBuilder) Test(ctx context.Context, src *dagger.Directory) (string, error) {
    return dag.Container().
        From("golang:1.24-alpine").
        WithDirectory("/src", src).
        WithWorkdir("/src").
        WithExec([]string{"go", "test", "./..."}).
        Stdout(ctx)
}

// BuildBackend cross-compiles the backend binary for Linux/amd64.
// Returns the compiled binary as a File.
func (m *WorkshopBuilder) BuildBackend(ctx context.Context, src *dagger.Directory) *dagger.File {
    return dag.Container().
        From("golang:1.24-alpine").
        WithDirectory("/src", src).
        WithWorkdir("/src").
        WithEnvVariable("CGO_ENABLED", "0").
        WithEnvVariable("GOOS", "linux").
        WithEnvVariable("GOARCH", "amd64").
        WithExec([]string{
            "go", "build",
            "-ldflags", "-s -w",
            "-o", "/out/workshop-backend",
            "./backend/",
        }).
        File("/out/workshop-backend")
}

// BuildWorkshop builds all step images for a workshop.
// workshopPath is relative to src (e.g. "examples/hello-linux").
func (m *WorkshopBuilder) BuildWorkshop(
    ctx context.Context,
    src *dagger.Directory,
    workshopPath string,
) error {
    // Get the backend binary
    backendBin := m.BuildBackend(ctx, src)

    // Get tool binaries (downloaded in Dagger, no local toolchain needed)
    tools := m.buildToolLayer(ctx)

    // Parse the workshop from the source tree
    // We do this by calling the shared library inside Go container
    workshopJSON, stepMetas, err := m.compileWorkshop(ctx, src, workshopPath)
    if err != nil {
        return fmt.Errorf("compiling workshop: %w", err)
    }

    // Build step images sequentially
    var prevImage *dagger.Container
    for i, step := range stepMetas {
        img, err := m.buildStepImage(ctx, src, workshopPath, step, i, workshopJSON, backendBin, tools, prevImage)
        if err != nil {
            return fmt.Errorf("building step %s: %w", step.ID, err)
        }

        tag := fmt.Sprintf("%s:%s", workshopJSON.Image, step.ID)
        fmt.Printf("Publishing %s\n", tag)
        _, err = img.Publish(ctx, tag)
        if err != nil {
            return fmt.Errorf("publishing %s: %w", tag, err)
        }
        prevImage = img
    }
    return nil
}
```

**Important design note:** The `compileWorkshop` step cannot directly call `pkg/workshop.Compile()` from Go inside the Dagger module itself (the Dagger module has its own go.mod). Instead, run a small Go binary inside a Go container that imports the shared library and outputs the compiled JSON. OR, more simply for MVP: do the compilation by shelling out to `go run` inside a container.

**Simpler approach for MVP:** Run the compilation step inside a Go container using `go run`:

```go
// compileWorkshop runs the workshop compiler inside a Go container.
// Returns workshop.json bytes and per-step metadata.
func (m *WorkshopBuilder) compileWorkshop(
    ctx context.Context,
    src *dagger.Directory,
    workshopPath string,
) (*workshopManifest, []stepMeta, error) {
    // Run a small Go script that imports pkg/workshop and prints JSON
    out, err := dag.Container().
        From("golang:1.24-alpine").
        WithDirectory("/src", src).
        WithWorkdir("/src").
        WithExec([]string{
            "go", "run", "./cmd/compile-workshop/",
            "--workshop", workshopPath,
        }).
        Stdout(ctx)
    if err != nil {
        return nil, nil, err
    }
    // Parse the JSON output
    // ... see cmd/compile-workshop/main.go below
}
```

This requires a small helper command at `cmd/compile-workshop/main.go` in the root module.

## `cmd/compile-workshop/main.go`

A thin CLI wrapper around `pkg/workshop.Compile()` that outputs JSON to stdout. Used by the Dagger pipeline.

```go
package main

import (
    "encoding/json"
    "flag"
    "fmt"
    "log"
    "os"
    "path/filepath"

    "github.com/asocpro/workshop-builder/pkg/workshop"
)

// Output is the JSON structure emitted to stdout.
type Output struct {
    WorkshopJSON string                `json:"workshopJson"`
    Steps        []StepOutput          `json:"steps"`
}

type StepOutput struct {
    ID           string `json:"id"`
    MetaJSON     string `json:"metaJson"`
    LLMJson      string `json:"llmJson,omitempty"`
    HasGoss      bool   `json:"hasGoss"`
    HasHints     bool   `json:"hasHints"`
    HasExplain   bool   `json:"hasExplain"`
    HasSolve     bool   `json:"hasSolve"`
    HasLLMDocs   bool   `json:"hasLlmDocs"`
}

func main() {
    workshopPath := flag.String("workshop", "", "path to workshop directory (relative to repo root)")
    flag.Parse()

    if *workshopPath == "" {
        log.Fatal("--workshop is required")
    }

    // Resolve to absolute path
    abs, err := filepath.Abs(*workshopPath)
    if err != nil {
        log.Fatalf("resolving path: %v", err)
    }

    // Parse and validate
    loaded, err := workshop.Parse(abs)
    if err != nil {
        log.Fatalf("parsing workshop: %v", err)
    }
    if err := workshop.Validate(loaded); err != nil {
        log.Fatalf("validating workshop: %v", err)
    }

    // Compile
    compiled, err := workshop.Compile(loaded)
    if err != nil {
        log.Fatalf("compiling workshop: %v", err)
    }

    // Build output
    out := Output{
        WorkshopJSON: string(compiled.WorkshopJSON),
    }
    for i, cs := range compiled.Steps {
        s := loaded.Steps[i]
        so := StepOutput{
            ID:         cs.ID,
            MetaJSON:   string(cs.MetaJSON),
            HasGoss:    s.HasGoss,
            HasHints:   s.HasHints,
            HasExplain: s.HasExplain,
            HasSolve:   s.HasSolve,
            HasLLMDocs: s.HasLLMDocs,
        }
        if cs.LLMJSON != nil {
            so.LLMJson = string(cs.LLMJSON)
        }
        out.Steps = append(out.Steps, so)
    }

    if err := json.NewEncoder(os.Stdout).Encode(out); err != nil {
        log.Fatalf("encoding output: %v", err)
    }
}
```

package main

import (
    "context"
    "encoding/json"
    "fmt"
    "strings"

    "dagger.io/dagger"
)

type WorkshopBuilder struct{}

// ── Test ─────────────────────────────────────────────────────────────────────

        WithDirectory("/src", src).
        WithWorkdir("/src").
        WithExec([]string{"go", "test", "-v", "./..."}).

        WithMountedCache("/root/.cache/go-build", dag.CacheVolume("go-build")).

// ── BuildWorkshop ─────────────────────────────────────────────────────────────

// BuildWorkshop builds all step OCI images for a workshop.
// workshopPath is relative to src (e.g. "examples/hello-linux").
func (m *WorkshopBuilder) BuildWorkshop(
    ctx context.Context,
    // +defaultPath="/"
    src *dagger.Directory,
    // Path to workshop directory relative to src root (e.g. "examples/hello-linux")
    workshopPath string,
) error {
    // Step 1: Compile workshop metadata via Go container
    compileOut, err := m.runCompileWorkshop(ctx, src, workshopPath)
    if err != nil {
        return fmt.Errorf("compile workshop: %w", err)
    }

    // Step 2: Build backend binary
    backendBin := m.BuildBackend(ctx, src)

    // Step 3: Download tool binaries
    tini := m.downloadTini(ctx)
    goss := m.downloadGoss(ctx)
    ttyd := m.downloadTtyd(ctx)

    // Step 4: Build step images sequentially
    var prev *dagger.Container

    for i, step := range compileOut.Steps {
        fmt.Printf("Building step %d/%d: %s\n", i+1, len(compileOut.Steps), step.ID)

        img := m.buildStepImage(ctx, src, workshopPath, compileOut, step, i, backendBin, tini, goss, ttyd, prev)

        tag := fmt.Sprintf("%s:%s", compileOut.WorkshopImage, step.ID)
        fmt.Printf("Publishing %s\n", tag)
        if _, err := img.Publish(ctx, tag); err != nil {
            return fmt.Errorf("publishing %s: %w", tag, err)
        }

        prev = img
    }
    return nil
}

// compileOutput mirrors the JSON from cmd/compile-workshop
type compileOutput struct {
    WorkshopJSON  string       `json:"workshopJson"`
    WorkshopImage string       `json:"workshopImage"` // populated after parsing workshopJson
    Steps         []stepOutput `json:"steps"`
}

type workshopJsonPartial struct {
    Image string `json:"image"`
}

type stepOutput struct {
    ID         string `json:"id"`
    MetaJSON   string `json:"metaJson"`
    LLMJson    string `json:"llmJson,omitempty"`
    HasGoss    bool   `json:"hasGoss"`
    HasHints   bool   `json:"hasHints"`
    HasExplain bool   `json:"hasExplain"`
    HasSolve   bool   `json:"hasSolve"`
    HasLLMDocs bool   `json:"hasLlmDocs"`
}

func (m *WorkshopBuilder) runCompileWorkshop(ctx context.Context, src *dagger.Directory, workshopPath string) (*compileOutput, error) {
    out, err := dag.Container().
        From("golang:1.24-alpine").
        WithMountedCache("/go/pkg/mod", dag.CacheVolume("go-mod")).
        WithDirectory("/src", src).
        WithWorkdir("/src").
        WithExec([]string{
            "go", "run", "./cmd/compile-workshop/",
            "--workshop", workshopPath,
        }).
        Stdout(ctx)
    if err != nil {
        return nil, err
    }

    var result compileOutput
    if err := json.Unmarshal([]byte(out), &result); err != nil {
        return nil, fmt.Errorf("parsing compile output: %w", err)
    }

    // Extract image name from workshop.json
    var wj workshopJsonPartial
    if err := json.Unmarshal([]byte(result.WorkshopJSON), &wj); err != nil {
        return nil, fmt.Errorf("parsing workshop.json image: %w", err)
    }
    result.WorkshopImage = wj.Image

    return &result, nil
}

func (m *WorkshopBuilder) buildStepImage(
    ctx context.Context,
    src *dagger.Directory,
    workshopPath string,
    compiled *compileOutput,
    step stepOutput,
    position int,
    backendBin *dagger.File,
    tini, goss, ttyd *dagger.File,
    prev *dagger.Container,
) *dagger.Container {
    var base *dagger.Container
    if prev == nil {
        // First step: start from Ubuntu 24.04
        base = dag.Container().From("ubuntu:24.04")
        // Install system deps
        base = base.WithExec([]string{
            "sh", "-c",
            "apt-get update && apt-get install -y --no-install-recommends " +
                "bash curl ca-certificates && rm -rf /var/lib/apt/lists/*",
        })
        // Install platform binaries
        base = base.
            WithFile("/sbin/tini", tini, dagger.ContainerWithFileOpts{Permissions: 0755}).
            WithFile("/usr/local/bin/goss", goss, dagger.ContainerWithFileOpts{Permissions: 0755}).
            WithFile("/usr/local/bin/ttyd", ttyd, dagger.ContainerWithFileOpts{Permissions: 0755}).
            WithFile("/usr/local/bin/workshop-backend", backendBin, dagger.ContainerWithFileOpts{Permissions: 0755})

        // Inject workshop-platform.bashrc
        base = base.WithNewFile("/etc/workshop-platform.bashrc", bashrcContent())

        // Source it from /etc/bash.bashrc
        base = base.WithExec([]string{
            "sh", "-c",
            `echo 'source /etc/workshop-platform.bashrc' >> /etc/bash.bashrc`,
        })

        // Bake /workshop/ metadata (ALL steps) — only in first step image
        base = m.bakeWorkshopMetadata(ctx, base, src, workshopPath, compiled)

        // Create runtime dir
        base = base.WithExec([]string{"mkdir", "-p", "/workshop/runtime"})
    } else {
        base = prev
    }

    // Apply this step's file mappings
    stepSrcDir := src.Directory(workshopPath + "/steps/" + step.ID + "/files")
    // For each file in step.yaml files[], we need to know source→target mapping.
    // We get this from a separate command or embed it in compileOutput.
    // For MVP: read the step.yaml inside the container to get file mappings.
    // Simpler: add file mappings to compileOutput in cmd/compile-workshop.
    // See note below — add FileMappings to compileOutput.

    // For now: copy all files/ contents into /workspace/ (simplified MVP)
    base = base.WithDirectory("/workspace", stepSrcDir)

    // Run step commands (also needs to come from compile output)
    // TODO: add commands to compileOutput

    // Entrypoint
    base = base.
        WithEntrypoint([]string{"/sbin/tini", "--", "/usr/local/bin/workshop-backend"})

    return base
}

func (m *WorkshopBuilder) bakeWorkshopMetadata(
    ctx context.Context,
    ctr *dagger.Container,
    src *dagger.Directory,
    workshopPath string,
    compiled *compileOutput,
) *dagger.Container {
    // workshop.json
    ctr = ctr.WithNewFile("/workshop/workshop.json", compiled.WorkshopJSON)

    // per-step metadata
    for _, step := range compiled.Steps {
        stepBase := "/workshop/steps/" + step.ID + "/"

        // meta.json
        ctr = ctr.WithNewFile(stepBase+"meta.json", step.MetaJSON)

        // llm.json
        if step.LLMJson != "" {
            ctr = ctr.WithNewFile(stepBase+"llm.json", step.LLMJson)
        }

        // Copy content files from source
        stepSrcPath := workshopPath + "/steps/" + step.ID + "/"
        stepSrc := src.Directory(stepSrcPath)

        // content.md (always present)
        ctr = ctr.WithFile(stepBase+"content.md", stepSrc.File("content.md"))

        // Optional convention files
        if step.HasGoss {
            ctr = ctr.WithFile(stepBase+"goss.yaml", stepSrc.File("goss.yaml"))
        }
        if step.HasHints {
            ctr = ctr.WithFile(stepBase+"hints.md", stepSrc.File("hints.md"))
        }
        if step.HasExplain {
            ctr = ctr.WithFile(stepBase+"explain.md", stepSrc.File("explain.md"))
        }
        if step.HasSolve {
            ctr = ctr.WithFile(stepBase+"solve.md", stepSrc.File("solve.md"))
        }
        if step.HasLLMDocs {
            ctr = ctr.WithDirectory(stepBase+"llm-docs/", stepSrc.Directory("llm-docs"))
        }
    }

    return ctr
}

func bashrcContent() string {
    return `# Workshop Platform Shell Instrumentation
# Appended to /etc/bash.bashrc

__workshop_log_command() {
    local exit_code=$?
    local cmd
    cmd=$(history 1 | sed 's/^[ ]*[0-9]*[ ]*//')
    if [ -n "$cmd" ]; then
        printf '{"ts":"%s","cmd":%s,"exit":%d}\n' \
            "$(date -u +%Y-%m-%dT%H:%M:%SZ)" \
            "$(echo "$cmd" | python3 -c 'import json,sys; print(json.dumps(sys.stdin.read().rstrip()))')" \
            "$exit_code" \
            >> /workshop/runtime/command-log.jsonl 2>/dev/null || true
    fi
}

PROMPT_COMMAND="__workshop_log_command${PROMPT_COMMAND:+; $PROMPT_COMMAND}"
`
}

## Important: Extend `compileOutput` with File Mappings and Commands

The `cmd/compile-workshop/main.go` output needs to include file mappings and commands per step so the Dagger pipeline can apply them. Extend the `StepOutput` struct:

```go
// In cmd/compile-workshop/main.go
type FileMapping struct {
    Source string `json:"source"`
    Target string `json:"target"`
    Mode   string `json:"mode,omitempty"`
}

type StepOutput struct {
    ID           string        `json:"id"`
    MetaJSON     string        `json:"metaJson"`
    LLMJson      string        `json:"llmJson,omitempty"`
    HasGoss      bool          `json:"hasGoss"`
    HasHints     bool          `json:"hasHints"`
    HasExplain   bool          `json:"hasExplain"`
    HasSolve     bool          `json:"hasSolve"`
    HasLLMDocs   bool          `json:"hasLlmDocs"`
    Files        []FileMapping `json:"files,omitempty"`
    Commands     []string      `json:"commands,omitempty"`
    Env          map[string]string `json:"env,omitempty"`
}
```

Populate from `loaded.Steps[i].Spec.Files`, `.Spec.Commands`, `.Spec.Env`.

Then in `buildStepImage`, instead of copying all of `files/` to `/workspace/`, apply mappings individually:

```go
for _, fm := range step.Files {
    srcFile := stepSrcDir.File(fm.Source)
    perms := 0644
    if fm.Mode != "" {
        // parse octal mode
        fmt.Sscanf(fm.Mode, "%o", &perms)
    }
    ctr = ctr.WithFile(fm.Target, srcFile, dagger.ContainerWithFileOpts{Permissions: fs.FileMode(perms)})
}
for _, cmd := range step.Commands {
    ctr = ctr.WithExec([]string{"sh", "-c", cmd})
}
for k, v := range step.Env {
    ctr = ctr.WithEnvVariable(k, v)
}
```

## Key Decisions

- **Dagger module has its own `go.mod`** at `dagger/go.mod` — separate from repo root
- **`cmd/compile-workshop/`** is a thin CLI in the root module that wraps `pkg/workshop.Compile()` — called from Dagger via `go run`
- **No local Go/Node** — everything runs in Dagger containers
- **Tool binaries** (tini, goss, ttyd) downloaded from GitHub releases inside Dagger — check for latest versions when implementing
- **First step image builds the full base** (installs system packages, platform tools, bakes `/workshop/`) — subsequent steps layer on top
- **`/workshop/` metadata** is baked in the FIRST step image only — all subsequent step images inherit it via Docker layer inheritance
- **Entrypoint** is `["/sbin/tini", "--", "/usr/local/bin/workshop-backend"]`
- **`ubuntu:24.04`** as base for MVP (proper base images come in M12)
- **Image tags**: `localhost/hello-linux:step-1-intro`, etc.

## Dagger Version Notes

Check the current Dagger version with `dagger version` on the workstation. Use `dagger init` to bootstrap the module — it pins the right engine version in `dagger.json` automatically.

The API shown uses Dagger Go SDK 0.18.x patterns. Check the actual Dagger SDK docs for the current API if anything differs.

## Bashrc Note

The `PROMPT_COMMAND` hook uses `python3` for JSON escaping as a simple approach. If python3 is not available in ubuntu:24.04 minimal (it usually isn't by default), either install it or use `jq` or a simpler escape:

```bash
cmd=$(echo "$cmd" | sed 's/\\/\\\\/g; s/"/\\"/g')
printf '{"ts":"%s","cmd":"%s","exit":%d}\n' "$ts" "$cmd" "$exit_code"
```

The simple approach works for typical commands but breaks on commands with quotes. The Python approach is more correct. Install python3-minimal if needed.
